A Few Accomplishments

In the last year of my undergradute career in Electrical Engineering, I have amassed a goldmine of knowledge regarding hardware and software design. Through many long nights of coding and studying, I was able to complete some really interesting projects. Here are a few of the projects I enjoyed working on the most.

RoboTank: Autonomous Self-driving Tank

Built around the TI Stellaris LM3S8962. RoboTank was designed to be controlled remotely via Bluetooth as well as autonomously using distance sensors to be maneuvered around confined spaces. Drivers and software for the ADCs, PWMs, UART and other hardware used were all written from scratch for this project (C). FreeRTOS (real time operating system) was used to program the tank.

  • Report

Universal Asynchronous Receive/Transmitter

Designed a UART in Verilog and programmed the NIOS II microprocessor on a FPGA to run a C application to send and receive byte sized data over a serial link.

  • Report

Class AB Amplifier with Variable Gain

Built an audio amplifier using an Op-amp design with BJTs to amplify a signal from an iPhone to a speaker using approximately 1 Watt of power consumption

  • Report

AC/DC Coverter with Voltage Regulator

Built an AC/DC converter using a full bridge rectifier and boost topology design to convert 7.5VAC to a variable 10VDC to 20VDC.
